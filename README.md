# chess-engine
Creating an AI bot that plays chess using methods such as Minimax, Neural Networks etc. For comparison and testing Tests Jupyter notebook can be used. It provides a friendly UI to the end user. After cloning this repo, downloading dataset specified in `source.txt` file and installing necessary python packages such as `pytorch` and `tensorflow` in a python virtual environment user can play against different models such as `Minimax`, `Negamax` and and CNN models with pytorch and tensorflow.
